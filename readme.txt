== Description ==

This plugin adds the dailymile training log widget, dailymile mileage counter widget(large and small), dailymile badge widgeth(white, grey, blue).

== Installation ==

1. Upload `dailymile-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Register your profile at http://www.dailymile.com if you haven't already.
4. Under Settings->dailymile options and add your dailymile username.
